# git管理
1. git checkout -b xxx

5. git push origin xxx

# 规范
1. 目录、文件名使用单个字母或者短线分隔的小写字母命名规范
2. css的类名使用短线分隔的先小写字母命名规范，id使用小驼峰命名规范
3. 变量名、函数使用小驼峰命名规范
